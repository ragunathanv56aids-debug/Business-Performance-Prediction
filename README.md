# 📊 Predictive Analytics Dashboard

## 🔎 Overview
This project demonstrates **predictive analytics using historical business data**.  
It includes **time-series forecasting, regression analysis, and customer trend insights** visualized in **Power BI**.

## 📂 Dataset
- Historical data (36 months)
- Columns: Month, Revenue, Marketing Spend, Customer Count, New Customers, Churn Rate, Profit, Region, Product Category, Sales Channel

## ⚙️ Features
1. Revenue forecasting (line chart with confidence interval)
2. Marketing vs Revenue regression (scatter chart with trend line)
3. Customer growth & churn analysis (KPI cards + visuals)
4. Profit trend visualization (area chart)
5. Region, Category, and Channel segmentation (map, bar, pie charts)

## 📊 Visuals
- Revenue Forecast Chart  
- Profit Trend Area Chart  
- Customer Retention KPI  
- Region-wise Map Analysis  
- Product Category Bar Chart  
- Sales Channel Pie Chart  

## 🚀 How to Run
1. Open Power BI Desktop
2. Import dataset (`dataset.csv`)
3. Build visuals using Power BI charts
4. Arrange visuals into a professional dashboard
5. Save and publish to GitHub repository
